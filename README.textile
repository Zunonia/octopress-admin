h1. Octopress Admin

An easy admin interface built via rails to interact with a barebone octopress/jekyll website.

h2. Why?

Inspired by "https://github.com/zkarpinski/Jekyll-Admin":https://github.com/zkarpinski/Jekyll-Admin

h2. Install:

This is a simple rails 3 app, so cd into the app dir and

<pre>
  <code>
    $: bundle install       #install required gems
    $: rake db:create:all   #create dbs
    $: rake db:migrate      #migrate db
    $: rails s              #start app
  </code>
</pre>
